HTML Basics

Page Skeleton

A basic HTML page usually starts with <!DOCTYPE html> and includes <html>, <head>, and <body>. The visible content usually sits inside the body.

Semantic Layout

Tags like <header>, <nav>, <main>, and <footer> describe the purpose of each section. That helps both people and browsers.

Attributes

Attributes such as id, class, and href add extra details to elements. They often connect HTML to CSS or JavaScript.

CSS Basics

Selectors

A selector tells CSS which elements to style. For example, p targets all paragraphs, .btn targets elements with class "btn", and #main targets the element with id="main".

Classes vs IDs

Use a class for styles that might be reused. Use an id when youâ€™re styling (or referencing) a unique element.

Box Model

Every element is basically a box: content, padding, border, and margin. Adjusting those values gives you control over spacing.

JavaScript Basics

Variables

Variables store values so you can reuse them. Modern JavaScript usually uses let and const.

Functions

Functions wrap reusable logic. A simple example: function greet(name) { console.log("Hi " + name); }

Events

JavaScript can react to user actions like clicks. For example: button.addEventListener("click", doSomething);
