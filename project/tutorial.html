How to Play

  1. Go to the Play Game page.
  2. Click the Start Game button.
  3. Read the question and then click the answer you think is correct.
  4. Try to answer before the timer reaches zero.
  5. Read the explanation for each question before moving on.
  6. At the end, check the summary panel to see your score and level.

Where HTML, CSS, and JS are used

HTML (Structure)

The quiz page uses HTML to define the layout: the question text, answer buttons, and status area. For example, the empty div with id answer-buttons is where the JavaScript injects the answer choices.

CSS (Purple Theme)

The whole purple color scheme lives in css/main.css. It handles layouts, fonts, spacing, and the color of buttons and cards.

JavaScript (Game Logic)

The file js/game.js stores questions, starts the timer, and updates the score and level as you play.

About the Timer

Each question gives you a limited number of seconds. When the time runs out, the question is marked as incorrect and the explanation still appears so that you can learn from it.
