About the Project

Quantum Shift was built as a small learning project to combine HTML structure, CSS styling, and a bit of JavaScript logic into one purple-themed package.

The main idea is to make the first steps into web development feel less overwhelming. Instead of reading long notes, you can play through a round and then look at the code behind it.

Technologies Used

  • HTML5 semantic layout
  • CSS for layout and purple color scheme
  • Vanilla JavaScript for quiz logic and timer
  • Font Awesome icons
